# 📌 Contexto do Projeto

## 🧱 Stack
- Electron + TailwindCSS + React + JavaScript
- SqLite database

## 📁 Estrutura
- /components → componentes reutilizáveis
- /pages → rotas
- /services → integração com API

## 📏 Padrões
- Sempre usar hooks
- Evitar classes
- Componentes devem ser pequenos e reutilizáveis
- Nomeação em inglês

## 🎯 Regras importantes
- Nunca acessar diretamente o banco fora de /services
- Sempre tipar dados quando possível
- Usar loading e error states

## ⚠️ Problemas atuais
- Integração com Google Calendar instável
- Performance na listagem de dados
